Cassandra::Compression::Compressors::Snappy class

A compressor that uses the Snappy compression library.

Note
This compressor requires the snappy gem (v0.0.10 or later for JRuby support).
Note
No need to instantiate this class manually, use compression: :snappy option when calling Cassandra.connect and one will be created automatically for you.
